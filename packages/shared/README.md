# @lrgendie/shared

Symphony'nin protokol sözleşmesi: WS/REST mesajlarının zod şemaları ve ortak tipler.
`@lrgendie/core` (daemon) ile `@lrgendie/cli`/masaüstü arayüzünün konuştuğu TEK ortak
dil budur — şemasız bir mesaj gönderilemez.

Bu paket doğrudan kurulmaz; `@lrgendie/cli`'nin bağımlılığıdır. Kaynak kod ve tam
dokümantasyon: https://github.com/lrgendie/symphony
